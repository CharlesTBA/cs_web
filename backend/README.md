Quick start

1. Install dependencies

```bash
cd backend
npm install
```

```bash

```

API Endpoints
- GET /api/grades
- GET /api/grades/:id
- POST /api/grades
- PUT /api/grades/:id
- DELETE /api/grades/:id

- GET /api/subjects
- GET /api/subjects/:id
- POST /api/subjects
- PUT /api/subjects/:id
- DELETE /api/subjects/:id

Notes
- Data is stored in `backend/data/*.json` for this scaffold. For production use replace with a real DB.
